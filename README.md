# 🎮 Desenvolvimento de Jogos 2D com Godot

Site educacional desenvolvido para auxiliar no aprendizado de desenvolvimento de jogos 2D utilizando a **Godot Engine**.

O conteúdo é apresentado de forma progressiva, através de aulas divididas em passos, permitindo que o aluno acompanhe cada etapa do desenvolvimento de um jogo.

## 🛠️ Tecnologias

* HTML
* CSS
* JavaScript
* Godot Engine
* GDScript

## 🎯 Objetivo

O projeto tem como objetivo tornar o aprendizado de desenvolvimento de jogos mais acessível, organizado e prático, apresentando os conteúdos de maneira gradual.

## 🚧 Status

Em desenvolvimento.

Novas aulas e conteúdos serão adicionados progressivamente.

## 📚 Finalidade

Este projeto está sendo desenvolvido para fins educacionais.
